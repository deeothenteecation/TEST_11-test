Projects >> FOC Loss Budget Calculator

Project Information

Category : Application

Project Date : October 2023

Tools Used : VS Code

Coding : Python, Flask, Flask Web Gui, HTML, CSS


Description:

Loss budget assessment refers to the computation of the approximate fiber optic cable system link loss performance characteristics. A loss budget analysis needs to be performed before building or installing a optical fiber cabling framework to ensure that the system will work over the intended link.

With this app, one can calculate the fiber optic cabling system loss budget by supplying and selecting required data into the proper fields. Standard values (fiber loss, connector, splice loss) for computing loss budget are based from the fiber optic association.
